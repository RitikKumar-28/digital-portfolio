## ✨ Features

- **Modern UI/UX:** A sleek, premium dark-themed design with smooth animations.
- **Interactive Elements:** Features custom CSS glassmorphism, subtle hover zoom effects, and dynamic gradients.
- **Projects Showcase:** Clean 16:9 aspect ratio cards detailing my cybersecurity engineering projects (including Password Strength Checker, Secure File Encryption, and AI Ransomware Detection).
- **Certifications Showcase:** A "framed document" layout highlighting my official credentials from Red Hat, Google, AWS, Oracle, and TryHackMe.
- **Responsive:** Fully responsive grid layout optimized for desktops, tablets, and phones.
- **Performance:** Built on Vite for lightning-fast HMR and optimized production builds.

## 🛠️ Technology Stack

- **Framework:** [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Styling:** Vanilla CSS with custom design system, variables, and glassmorphism.

## 💻 Running Locally

To run this project on your local machine, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/RitikKumar-28/digital-portfolio.git
   cd digital-portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173`.
